# MAT Reader Test Data

This directory should contain the MATLAB v7.3 test files generated by `generate_test_files.m`.

## Setup Instructions

1. **Generate Test Files in MATLAB**:

   ```matlab
   % Navigate to the test/ directory
   cd path/to/dartframe/test

   % Run the generator script
   generate_test_files
   ```

2. **Copy Files to This Directory**:
   The script will create these files in the current MATLAB directory:

   - `test_numeric.mat` - Various numeric types
   - `test_strings.mat` - Character and string arrays
   - `test_logical.mat` - Boolean/logical arrays
   - `test_cells.mat` - Cell arrays
   - `test_structures.mat` - Structure arrays
   - `test_complex.mat` - Complex mixed data
   - `test_edge_cases.mat` - Edge cases
   - `test_multidim.mat` - Multidimensional arrays

3. **Run Tests**:
   ```bash
   dart test test/mat_reader_test.dart
   ```

## File Descriptions

### test_numeric.mat

Contains various numeric data types:

- Scalars (double, int)
- Vectors (row and column)
- 2D matrices (small and large)
- All integer types (int8, uint8, int16, ... int64, uint64)
- Single and double precision floats

### test_strings.mat

Contains string and character data:

- Character scalars
- Character arrays
- 2D character matrices
- String arrays (if supported by MATLAB version)

### test_logical.mat

Contains boolean/logical data:

- Logical scalars
- Logical vectors
- Logical matrices
- Logical arrays from comparisons

### test_cells.mat

Contains cell arrays:

- Simple cell arrays
- Mixed type cells
- 2D cell arrays
- Nested cell arrays
- Cell arrays with matrices

### test_structures.mat

Contains structure data:

- Simple structures
- Nested structures
- Structure arrays
- Structures with mixed field types

### test_complex.mat

Contains realistic complex data:

- Experiment data with nested structures
- Large matrices
- Large cell arrays with matrix elements

### test_edge_cases.mat

Contains edge cases and special values:

- Empty arrays (double, char, cell, struct)
- Special numeric values (inf, -inf, nan)
- Very small and large values
- Single element arrays

### test_multidim.mat

Contains multidimensional arrays:

- 3D arrays
- 4D arrays
- 5D arrays

## Note

If you don't have MATLAB, you can use GNU Octave (free alternative):

```bash
octave generate_test_files.m
```

However, some features (like string arrays) may not be available in Octave.
